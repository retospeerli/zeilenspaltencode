# Zeilen-Spalten-Code

Eine kleine GitHub-Pages-App für den Zeilen-Spalten-Code nach der gezeigten Unterrichtsvorlage.

## Funktionen

- frei wählbares Passwort / Schlüsselwort
- automatische Bildung eines 5-Buchstaben-Schlüssels
- I und J werden zusammengelegt
- Verschlüsselung und Entschlüsselung
- verschlüsselter Text wird in 5er-Gruppen dargestellt
- Code-Tabelle wird live angezeigt
- Beispieltext und Kopieren-Button

## Nutzung auf GitHub Pages

1. Repository erstellen.
2. Alle Dateien hochladen.
3. In GitHub unter **Settings → Pages** die Quelle **main branch / root** wählen.
4. App öffnen.

## Hinweis zum Schlüssel

Der historische Zeilen-Spalten-Code braucht fünf verschiedene Buchstaben als Zeilen- und Spaltenköpfe. Diese App nimmt aus dem frei gewählten Passwort die ersten fünf unterschiedlichen Buchstaben. Falls es weniger als fünf sind, wird mit weiteren Alphabetbuchstaben ergänzt.
